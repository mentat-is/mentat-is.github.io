What is it?

Graphical Universal Log Processor, gULP for short, is an open source log parser and visualizer built from the ground up by Mentat!
It features an innovative UI and a lot of useful feature for all you incident response, threat hunting and forensics needs!

Get it now!

Simply copy-paste the code below inside your terminal to get rolling!

curl https://raw.githubusercontent.com/mentat-is/gulp/refs/heads/develop/setup.sh | sudo bash

Features

Fancy UI

Gulp has an innovative and data-driven UI, built by analysts for analysts.

Co-op / Collaboration

Easily collaborate with others in your team, leave notes, link events and catch the bad guys, together!

Sigma Rules

Built-in support for Sigma rules, automate detection of known attacks easily!

Extensible

Extend APIs, supported log formats and more with easy to write plugins!
